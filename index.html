Collaborative Annotation

MicroDraw is a web application to visualise and annotate collaboratively high resolution histology data. Annotations are vectorial, resolution independent, and allow for boolean operations to combine, subtract and split regions. Different ontologies are available to produce collaborative, community curated atlases, and to annotate staining artefacts to simplify the automatic processing and image analyses of the data. All modifications are saved into a common repository based on git, allowing for forking and branching. MicroDraw's RESTful API allows users to access annotation data for offline processing.
